## [0.1.0] - 2024-01-01

### Added

- 初始版本发布
- LangGraph 工作流编排框架
- Milvus 向量检索服务集成
- MySQL 异步数据存储
- FastAPI REST API
- Docker 和 Kubernetes 部署支持
- 支持的异常类型:
  - Checkpoint 失败
  - 背压问题
  - 反序列化错误
  - OOM 问题
  - 网络问题

### Infrastructure

- GitHub Actions CI/CD 流程
- Pre-commit 代码质量检查
- Dependabot 依赖更新
- CodeQL 安全扫描
